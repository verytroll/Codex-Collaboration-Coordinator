## When To Use The Archive

Open these files only when you need:

- historical product reasoning
- dependency tracing across phases
- release packaging tied to an older baseline
- evidence that a design or workflow predates the active phase

For current work, go back to [planning index](../INDEX.md) and start from the active phase docs.
