## Read Only What You Need

- product or roadmap question: open `PRD.md`
- route or payload question: open `API.md`
- repository, service, or orchestration design question: open `ARCHITECTURE.md`
- migration or persistence question: open `DB_SCHEMA.md`
- historical inspiration question: open `BORROWED_IDEAS.md`

Do not load all reference docs together unless the task genuinely spans multiple contracts.
