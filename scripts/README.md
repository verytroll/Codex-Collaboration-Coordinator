# Scripts

Local helper scripts:

- `run.ps1` start the app with uvicorn
- `dev.ps1` start the app with the development defaults
- `seed.ps1` apply migrations and seed demo data
- `smoke.ps1` validate health, agent card, seed data, and basic message flow while waiting for the app to become ready
- `test.ps1` run the test suite
- `lint.ps1` run Ruff checks
